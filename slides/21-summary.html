Key Takeaways

The Performance Mindset

  1. Measure first — don't guess where bottlenecks are
  2. Reduce allocations — every object is a GC promise; use opaque data, object reuse, pool allocators
  3. Avoid unnecessary async — sync fast paths, async return pattern
  4. Pool expensive resources — timers, buffers, connections
  5. Yield cooperatively — keep the event loop responsive
  6. Schedule with priorities — not all work is equal
  7. Use shared memory — SharedArrayBuffer over MessagePort
  8. Configure your runtime — UV thread pool, semi-space, heap size, Buffer.poolSize, stream.defaultHighWaterMark
