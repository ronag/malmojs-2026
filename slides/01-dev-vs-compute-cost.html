Developer Cost vs Compute Cost

The most important optimization decision

The Economics

  • Senior developer: ~$150k–250k/year
  • AWS EC2 instance (c7g.xlarge): ~$1,100/year
  • One developer ≈ cost of 150 servers
  • "Premature optimization is the root of all evil" — Knuth
    But knowing where to optimize is the root of all good.
The point isn't "don't optimize." The point is: optimize the right things. When your code runs on 100+ servers, saving 10% CPU saves real money. When it runs on 2 servers, spend that time on features instead.

When to Optimize

  • Hot paths — code that runs millions of times/sec
  • High scale — 100+ servers, every % matters
  • Latency-sensitive — real-time, trading, media
  • Resource-constrained — edge, IoT, embedded
Don't optimize code that runs once per request on 2 servers. Measure first, then decide.
